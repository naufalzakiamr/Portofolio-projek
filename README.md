## ğŸ“ Customization

### Personal Information
Edit `src/data/personal.js` untuk mengubah:
- Nama dan title

- Menambah/mengurangi skill categories
- Update skill levels

- Menambah project baru
- Update project details
- Ganti images dan links

### Styling
Edit `src/index.css` untuk:
- Mengubah color palette
- Menambah animations baru
- Customize components
