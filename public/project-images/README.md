# Project Images

Letakkan foto-foto project Anda di folder ini dengan nama file yang sesuai:

## Nama File yang Diperlukan:
- `ecommerce.jpg` - Foto untuk E-Commerce Platform
- `taskmanager.jpg` - Foto untuk Task Management App  
- `weather.jpg` - Foto untuk Weather Dashboard
- `lms.jpg` - Foto untuk Learning Management System
- `realestate.jpg` - Foto untuk Real Estate Platform
- `portfolio.jpg` - Foto untuk Portfolio Website

## Format yang Disarankan:
- Format: JPG, PNG, atau WebP
- Ukuran: 500x300 pixels (rasio 5:3)
- Ukuran file: Maksimal 500KB untuk performa optimal

## Cara Mengganti Foto:
1. Siapkan foto project Anda
2. Rename sesuai nama file di atas
3. Copy ke folder `public/project-images/`
4. Foto akan otomatis muncul di halaman Projects

## Fallback:
Jika foto tidak ditemukan, akan menampilkan placeholder atau error. Pastikan nama file sesuai dengan yang ada di `src/data/projects.js`.
